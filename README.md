# Simple FPS Shooting Game 

This is a basic First-Person Shooting (FPS) game made with HTML, CSS, and JavaScript.  
You control a crosshair with your mouse and shoot the moving targets.  
The game also has a small aim-assist feature that helps you aim near the target.

---

##  Game Features
- Mouse-controlled crosshair  
- Moving targets on the screen  
- Click to shoot the target  
- Score increases on every hit  
- 60-second timer  
- Game Over screen  
- Simple aim-assist (cursor moves slightly toward nearby targets)

---

##  Project Files
- index.html
- style.css
- script.js
- README.md

  
---

##  How to Play
1. Move your mouse to control the crosshair  
2. When a target comes in front, click to shoot it  
3. Your score will increase  
4. After 60 seconds, the game will stop  
5. Your final score will be shown  

---

##  Tools Used
- HTML  
- CSS  
- JavaScript  

---

## How to Run
1. Download all files  
2. Open **index.html** in your browser  
3. The game will start automatically
